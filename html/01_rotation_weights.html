Understanding Linear Classification with Weights

Some fruits are safe to eat while others are poisonous. There is a pattern: size of spots and length of spikes. We need needs cognitive decision for categorizing. , most basic Neural Network with only inputs and outputs.

This is what a simple neural network does! It takes these two measurements as inputs, multiplies them by weights, and produces outputs that tell us the classification. The decision boundary (the line separating safe from poisonous) can be rotated by adjusting the weights.

See the neural network in action. The blue region represents the network predicts fruits are safe, and red region predicts they're poisonous. Use the weight sliders to rotate the decision boundary a full 360 degrees!

Decision Rule:

output1 = input1 × w1,1 + input2 × w2,1

output2 = input1 × w1,2 + input2 × w2,2

If output1 > output2 → Safe | If output2 > output1 → Poisonous

The Classification Code

// Weight values connecting each input to the outputs
let weight_1_1, weight_2_1;  // → output_1 (safe)
let weight_1_2, weight_2_2;  // → output_2 (poisonous)

// Classifies the given fruit as either safe (0) or poisonous (1)
function classify(input_1, input_2) {
    const output_1 = input_1 * weight_1_1 + input_2 * weight_2_1;
    const output_2 = input_1 * weight_1_2 + input_2 * weight_2_2;

    // Return 0 if output_1 is greater (safe), otherwise return 1 (poisonous)
    return (output_1 > output_2) ? 0 : 1;
}

// Visualize by running classify on every pixel in the graph
function visualize(graphX, graphY) {
    const predictedClass = classify(graphX, graphY);

    if (predictedClass === 0) {
        setColor(graphX, graphY, safeColor);     // Blue
    } else {
        setColor(graphX, graphY, poisonousColor); // Red
    }
}
